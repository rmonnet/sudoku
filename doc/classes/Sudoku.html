Class Sudoku
In: sudoku.rb
Parent: Object

The main class for Sudoku. It contains a problem (9x9 grid) partially solved and associated data such as the list of all possible values for all locations.

In Ruby’s tradition, grid index starts at 0.

Sudoku.new([[1,1,9]], parent) → Sudoku problem similar to parent but with value 9 filled-in at location (1,1)

create a new problem. If a parent is given, then the new problem starts with the parent values, if not all locations are empty. The list of partial solution is then added to the problem. Each solution is described as a triplet [row, col, value].
